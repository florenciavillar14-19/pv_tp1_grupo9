Nombre del Proyecto: Sistema de Gestión de Tareas Académicas.

Fecha de Inicio:

05/09/2023

Descripción del Proyecto:

El proyecto de Programación Visual consiste en el desarrollo de una aplicación de escritorio orientada a la gestión de tareas académicas, permitiendo a los usuarios registrar, modificar y eliminar actividades de manera intuitiva mediante una interfaz gráfica. La aplicación fue diseñada utilizando componentes visuales como botones, cuadros de texto y listas, facilitando la interacción del usuario sin necesidad de conocimientos técnicos avanzados. Además, se implementaron validaciones de datos para garantizar la correcta carga de información y se organizaron los elementos de la interfaz siguiendo criterios de usabilidad y accesibilidad. El objetivo principal del proyecto es aplicar los conceptos fundamentales de la programación orientada a eventos, comprendiendo el funcionamiento de las interfaces gráficas y la interacción usuario-sistema. Como objetivos específicos, se busca desarrollar habilidades en el diseño de interfaces amigables, gestionar correctamente eventos como clics y entradas de datos, y estructurar el código de manera clara y mantenible. Este proyecto permite integrar conocimientos teóricos con una aplicación práctica, simulando una herramienta útil en el ámbito educativo para la organización y seguimiento de tareas.
